# Как получить готовый APK через GitHub

## 1. Создайте аккаунт GitHub

Откройте **github.com** и зарегистрируйтесь, если аккаунта ещё нет.

## 2. Создайте новый проект

1. Нажмите значок **+** в правом верхнем углу.
2. Нажмите **New repository**.
3. В поле имени укажите: `MojKrvniTlak`.
4. Выберите **Private**.
5. Нажмите **Create repository**.

## 3. Загрузите файлы

1. На странице нового проекта нажмите **uploading an existing file**.
2. Откройте этот ZIP на телефоне и сначала извлеките его в отдельную папку.
3. Загрузите в GitHub всё содержимое папки `moj-krvni-tlak`, включая скрытую папку `.github`.
4. Внизу страницы нажмите **Commit changes**.

Важно: в корне GitHub должны сразу находиться папка `app`, файлы `build.gradle.kts`, `settings.gradle.kts` и папка `.github`. Не загружайте их внутрь дополнительной папки.

## 4. Запустите сборку

1. Откройте вкладку **Actions**.
2. Выберите слева **Build Android APK**.
3. Нажмите **Run workflow**.
4. Ещё раз нажмите зелёную кнопку **Run workflow**.
5. Подождите примерно 3–10 минут.

## 5. Скачайте приложение

1. Откройте завершённую сборку с зелёной галочкой.
2. Внизу найдите раздел **Artifacts**.
3. Нажмите **MojKrvniTlak-APK**.
4. Скачается ZIP-файл. Извлеките из него `MojKrvniTlak.apk`.
5. Откройте APK на телефоне и разрешите установку приложений из этого источника.

Это тестовая версия приложения. При установке Android может показать предупреждение, потому что APK не опубликован в Google Play.
